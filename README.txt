/**********  LANCER L'INTERFACE CONSOLE/GRAPHIQUE **********/

Voici les consignes pour pouvoir compiler et executer avec succès ce programme:

----> Premièrement si ce n'est pas déjà fait il faut installer la librairie SDL2, ainsi que SDL2_image :
sudo apt-get install sdl2-dev
sudo apt-get install libsdl2-image-dev


----> Puis compiler et executer en se mettant dans le répertoire général du projet :
make
./main_interface
